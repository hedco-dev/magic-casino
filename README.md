# Magic Casino

Applicant's test for Javascript coders.

## How to use

You can find more information [here](/docs).

## Setup

```
$ npm install
```

### Running

```
$ npm start
```

## Build

```
$ npm run build
```

## Document

```
$ npm run doc
```
